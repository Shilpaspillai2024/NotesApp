# MERN Notes App

A simple Notes application built using:

- **Backend:** Node.js + Express + MongoDB + Mongoose  
- **Frontend:** React + Vite + Tailwind CSS  
- **Database:** MongoDB Atlas or Local MongoDB

---

## Project Structure
notes_app/backend/->Node.js+Express+Mongoose
notes_app/frontend/->React Ui(Vite+Tailwind css)

## Backend Setup

```bash
cd backend
npm install

Create a .env file inside backend with:

PORT=5000
MONGO_URI=your_mongodb_connection_string

Run the API:

npm run dev


##Frontend Setup
cd frontend
npm install
npm run dev
